Interpréteur Lua 5.1 (Luau)

Plus couramment appelé « Script Builder Place » dans Roblox est une expérience où les joueurs peuvent écrire et exécuter leurs propres scripts Lua en temps réel. Elle offre un éditeur de code intégré, un environnement sécurisé (sandbox), et des outils pour tester des idées ou apprendre le développement dans Roblox, celle-ci possède de multiples fonctions personnalisables afin d’améliorer le confort des utilisateurs.

Extension Chrome

Ce projet est une extension chrome qui extrait le texte provenant d’images sur un site, le traduit et replace celui-ci sur le texte précédent afin de pouvoir lire dans le langage correct. Principalement utilisé pour lire des mangas (scans) en ligne, qui n’ont pas encore été traduit (pour les impatients).
